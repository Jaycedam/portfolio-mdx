# Portfolio V1 - Website

_React, Framer Motion, Django, PostgreSQL_

<HeaderImage
  src="https://github.com/Jaycedam/portfolio-mdx/raw/main/projects/portfolio-v1/preview.webp"
  alt="header image"
/>

Página web personal creada con React (CRA), realizada para mostrar proyectos destacados. El backend (Django) se encarga de generar la API con el listado de proyectos. Autorización con JWT es necesaria para todos lo requests excepto GET.

_Esta versión de la página es obsoleta en favor de la versión creada con Next.js_

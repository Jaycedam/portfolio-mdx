## Features

1. Install Homebrew.
2. Restore the current backup:
   - Configuration files (.config, .zprofile, .zshrc).
   - Homebrew-installed packages through a Brewfile.
3. Modify some macOS default preferences:
   - Dock: Disable auto-hide, disable recent applications.
   - Finder: Show file extensions, show hidden files, show path bar, default to list view, keep folders on top, scope search to current folder.
   - Spaces: Disable automatic Spaces reordering, group by application in Mission Control.
   - Security: Enable firewall.

## How to use

Get started

```bash
git clone https://github.com/Jaycedam/mac-setup.git && cd mac-setup && bash main.sh
```

Update backup

```bash
cd path-to-project
bash backup.sh
```

Then

```bash
git add .
git commit -m "chore: backup updated"
git push origin main
```

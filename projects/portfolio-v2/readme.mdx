# Portfolio V2

_Stack: Next.js con TypeScript, Tailwind CSS, Shadcn UI, Next-Auth, React-Hook-Form, Zod, Bcrypt, Framer Motion, Prisma, MySQL._

Portfolio personal serverless realizado para mostrar proyectos destacados. Utilización de Server Side Rendering (SSR) con ciertos componentes CSR que necesitan de mayor interactividad. Server Actions para la página de Admin. Incluye dark mode automático con toggle para sobreescribir el tema.

## Página principal

La página principal muestra un listado de 4 proyectos destacados con un bottón para ver el resto en una nueva ruta "/projects". También muestra el historial de mi carrera profesional como programador de software.

![screenshot](https://github.com/Jaycedam/portfolio-mdx/raw/main/projects/portfolio-v2/projects.webp)

Todos estos datos son obtenidos desde una base de datos remota MySQL. El listado de proyectos posee el nombre, URL de la imágen del listado, URL a un archivo MDX remoto que obtiene los detalles (como este archivo actual), área del proyecto y un boolean que determina si el proyecto es mostrado en la página principal.

## Página de Administrador

La página de administrador está protegida con next-auth y utiliza credenciales almacenadas en una base de datos MySQL. Las contraseñas se almacenan de manera segura mediante el uso de Bcrypt al registrar.

![screenshot](https://github.com/Jaycedam/portfolio-mdx/raw/main/projects/portfolio-v2/admin.webp)

### Características

- Sidebar: Esta página incluye una barra lateral con enlaces referentes a cada tabla de la base de datos, además de la página principal de admin.
- Tablas: Al hacer clic en un enlace de la barra lateral, se muestra una tabla con la lista de elementos en la base de datos. Cada columna tiene botones para eliminar o actualizar la entrada correspondiente, también existe un bottón para crear una nueva entrada, la cual se encuentra al lado del título.
